Breadcrumbs

Button styles can be applied to any element. Typically you'll want to use either a <button> or an <a> element:

  1. Step 1
  2. Step 2
  3. Step 3
  4. Step 4
<ol class="breadcrumb"> 
    <li>Step 1</li>
    <li>Step 2</li>
    <li>Step 3</li>
    <li class="active">Step 4</li>
</ol>
